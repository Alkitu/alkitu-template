# Alkitu Media Service Images - Style Guidelines

This document details the exact prompt characteristics used to generate the 3D service icons found in this `Services_IMG` directory. Use these guidelines to ensure any newly generated or manually created icons perfectly match the application's clean, premium, and distinct visual style.

## Core Prompt Template

To generate new icons that seamlessly match existing ones, use the exact base prompt template below, simply swapping out `[OBJECT / SUBJECT DESCRIPTION]` for the specific item you need:

> "A minimalist, high-quality, 3D rendered icon of **[OBJECT / SUBJECT DESCRIPTION]**. The icon should have a consistent visual style suitable for a modern home service app. The background should be a subtle, solid color (e.g., light gray or very pale blue). No text, only the object. Soft lighting, clean lines."

## Essential Aesthetic Characteristics

If you are communicating with a designer or building the icons manually, adhere to the following rules:

1. **Format Base**: Premium, Minimalist **3D Render**.
2. **Subject Matter**: The subject must be heavily focused, centered, and immediately recognizable, isolated from excessive clutter.
3. **Materials**: Soft, matte textures mixed with subtle glossy reflections where appropriate (e.g., clean ceramic tiles, metal faucets). Avoid photorealism; aim for a stylized but highly polished "modern app icon" look.
4. **Lighting**: Soft, diffused studio lighting. Avoid harsh shadows or extremely high contrast.
5. **Background**: Must be consistently a very subtle, pale, solid color. Specifically, very soft grays (`#F4F4F5`, `#FAFAFA`) or very pale cool blues. The background shouldn't distract from the object.
6. **Angle**: Isometric or slight top-down viewing angles generally work best to show volume and context of the object without completely flattening it.
7. **No Text**: Do not include any text, typography, letters, or numbers inside the image itself. 
8. **Edges**: Clean, thick outlines or smooth rounded geometry. Avoid excessively sharp, jagged, or extremely low-poly elements.

## Examples of Usage
- **For "Bathroom Vanity":** "...a modern bathroom vanity with a sink and a mirror above it."
- **For "Deep Cleaning":** "...a modern mop and a cleaning bucket with bubbles."
- **For "Floor Plan Service":** "...a rolled out architectural floor plan or blueprint featuring a simple house layout."

Following these guidelines will maintain absolute visual harmony across the entire suite of services.
