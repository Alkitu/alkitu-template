## Description

**URL:** `/app/requests/new/location`

**User Story Intro:**

Como cliente, quiero seleccionar o crear una ubicación para indicar dónde debe realizarse el servicio.

**Funciones:**

* **Ver mis ubicaciones guardadas**

    \[GET /app/locations\]


* **Crear nueva ubicación**

    \[POST /app/locations\]


* **Seleccionar ubicación y continuar**

    \[→ `/app/requests/new/service`\]
